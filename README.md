# 🚛 BaySync V2

Fork mejorado de BaySync — Control de muelles + registro digital de camiones.

## 🌐 URLs

- **BALIE (tu PC):** https://sacharunaaa-max.github.io/baysync-v2/
- **Almacén (smartphone):** https://sacharunaaa-max.github.io/baysync-v2/encargado.html

## Flujo

1. Abres día → pestaña **📝 Registro** → ingresas conductor, proveedores, puerta, llave
2. El encargado abre **📋 BaySync V2 · Almacén** en su smartphone
3. Ve el camión entrante, toca "Registrar descarga", llena cantidades y firma
4. Tú ves el camión como "completado" en pestaña **🚛 En curso**
5. El conductor regresa → confirmas llave, despachas → puerta se libera
6. Fin del día → ⬇ **CSV** con todo listo

## Tecnología

Firebase Realtime Database + HTML/CSS/JS vanilla. Sin frameworks.
